# NightLight Frontend

This is the client-side React application for NightLight.

## 🚀 Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🛠 Deployment
This frontend is configured for deployment on **Netlify**.
Refere to the main project `README.md` in the root directory for full deployment instructions.
